Calidad del Software

Normativas aplicadas al Procesamiento del Lenguaje Natural

ISO/IEC 25010

Esta norma define un modelo de calidad para productos de software, con características como funcionalidad, usabilidad y mantenibilidad.

Aplicación en sistemas NLP:
  • Funcionalidad: Precisión en el procesamiento de lenguaje
  • Usabilidad: Interfaces conversacionales intuitivas
  • Eficiencia: Rendimiento en tiempo real
  • Mantenibilidad: Modularidad en componentes de NLP
  • Portabilidad: Adaptación a diferentes entornos
Ejemplo de implementación:

Un sistema de análisis de sentimientos debe ser evaluado regularmente en precisión (funcionalidad), tener una API bien documentada (usabilidad), procesar textos rápidamente (eficiencia), y permitir actualizaciones de modelos (mantenibilidad).

IEEE 830

El estándar IEEE 830 proporciona guías para la especificación de requisitos de software (SRS), crucial en proyectos complejos de NLP.

Requisitos clave para sistemas NLP:
  • Funcionales: Capacidades de procesamiento específicas
  • Interfaces: APIs, interfaces de usuario conversacionales
  • Rendimiento: Tiempos de respuesta, carga máxima
  • Restricciones: Idiomas soportados, formatos de entrada
  • Calidad: Precisión mínima requerida
Caso práctico:

Para un chatbot médico, los requisitos deben especificar exactamente qué tipos de consultas puede manejar, cómo debe responder a preguntas fuera de su dominio, y los protocolos para derivar a profesionales humanos cuando sea necesario.

WCAG 2.1

Las Pautas de Accesibilidad para el Contenido Web son esenciales para interfaces conversacionales accesibles.

Principios aplicados a NLP:
  • Perceptible: Alternativas para contenido de voz
  • Operable: Navegación por teclado en chatbots
  • Comprensible: Lenguaje claro y predecible
  • Robusto: Compatibilidad con tecnologías asistivas
Implementación accesible:

Un asistente virtual debe ofrecer: transcripción de respuestas de voz, controles compatibles con lectores de pantalla, opción para simplificar el lenguaje, y evitar dependencia exclusiva de reconocimiento de voz para usuarios con dificultades del habla.
